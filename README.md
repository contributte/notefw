## About NoteFw

NoteFw was an alternative Nette-inspired micro-tiny framework. This monorepo contained the core framework components:

- **note/app** (`note/`) - Core framework with DI container and HTTP handling
- **note/router-nette** (`router-nette/`) - Nette-based router component
- **note/view-latte** (`view-latte/`) - Latte template engine integration
- **note/project** (`project/`) - Simple project skeleton

### Framework Components

#### Core (note/app)
The core framework component provided:
- Dependency injection container (based on Nette DI)
- HTTP request/response handling
- PSR-4 autoloading

#### Router (note/router-nette)
Router component based on Nette Framework's routing system.

#### View (note/view-latte)
Template rendering using the Latte template engine.

#### Project Skeleton
A simple project template that integrated all framework components together.
